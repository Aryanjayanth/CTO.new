# Prodify - Quick Start Guide

Get up and running with Prodify in under 5 minutes!

## üöÄ Installation

### Option 1: Run Locally (Development)

```bash
# Clone the repository
git clone <your-repo-url>
cd prodify

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will open at `http://localhost:5173`

### Option 2: Build for Production

```bash
# Install dependencies
npm install

# Build the app
npm run build

# Preview the production build
npm run preview
```

### Option 3: Deploy to Web

See [DEPLOYMENT.md](DEPLOYMENT.md) for hosting on Vercel, Netlify, or other platforms.

## üìù First Steps

### 1. Create Your First Task

1. Click **"Tasks"** in the sidebar (should be selected by default)
2. Click the **"+ New Task"** button
3. Fill in:
   - **Title**: "Welcome to Prodify" (required)
   - **Description**: Add details about the task
   - **Priority**: Choose Low, Medium, or High
   - **Due Date**: Set a deadline
4. Click **"Create"**

‚úÖ Your first task is now visible in the task list!

### 2. Add a Note

1. Click **"Notes"** in the sidebar
2. Click the **"+ New Note"** button
3. The note editor opens with "Untitled Note"
4. Change the title by clicking on it
5. Type your content in the editor
6. Add a folder name (e.g., "Personal")
7. Add tags by typing and pressing Enter
8. Click **"Save"** (or changes auto-save)

üìù Your note is now saved and searchable!

### 3. Schedule an Event

1. Click **"Calendar"** in the sidebar
2. Click **"+ New Event"** button
3. Fill in:
   - **Title**: Name of your event
   - **Start Time**: When it begins
   - **End Time**: When it ends (optional)
   - **Location**: Where it takes place
   - **Type**: Work, Personal, Health, or Other
4. Click **"Create"**

üìÖ Your event appears on the calendar!

### 4. Track a Habit

1. Click **"Habits"** in the sidebar
2. Click **"+ New Habit"** button
3. Enter:
   - **Habit Name**: "Morning Exercise"
   - **Frequency**: Daily or Weekly
   - **Color**: Pick your favorite
4. Click **"Create"**
5. Click on today's date to mark it complete

üéØ Your habit tracking begins!

### 5. Plan Your Day

1. Click **"Timetable"** in the sidebar
2. Click **"+ New Routine"** button
3. Set up:
   - **Title**: "Morning Workout"
   - **Start Time**: 07:00
   - **End Time**: 08:00
   - **Days**: Select Mon, Wed, Fri
   - **Category**: Health
   - **Color**: Green
4. Click **"Create"**

‚è∞ Your routine appears in the schedule!

## üí° Tips for Success

### Task Management
- **Use Priorities Wisely**: Not everything can be high priority
- **Break Down Big Tasks**: Use subtasks for complex projects
- **Set Realistic Due Dates**: Give yourself enough time
- **Review Daily**: Check your task list each morning

- **Use Consistent Folders**: Stick to a few main categories
- **Tag Everything**: Tags make searching easier
- **Write Clear Titles**: You'll thank yourself later
- **Regular Review**: Go through old notes monthly

### Habit Tracking
- **Start Small**: Begin with 2-3 habits maximum
- **Be Consistent**: Track at the same time each day
- **Don't Break the Chain**: Maintain your streaks
- **Review Weekly**: Check your success rates

### Calendar Planning
- **Color Code Events**: Use types to organize visually
- **Add Locations**: Helps with planning your day
- **Set Reminders**: Note important details
- **Review Weekly**: Plan your upcoming week

### Timetable
- **Be Realistic**: Don't over-schedule
- **Leave Buffer Time**: Don't pack every minute
- **Match Energy Levels**: Hard tasks when you're fresh
- **Stay Flexible**: Life happens, adjust as needed

## üîß Troubleshooting

### App Not Loading?
- Clear browser cache
- Try a different browser
- Check console for errors (F12)

### Data Not Saving?
- Check localStorage is enabled
- Clear old data if storage is full
- Export and reimport data

### Features Not Working?
- Refresh the page
- Clear localStorage and restart
- Check browser compatibility

### Need to Reset?
1. Open browser DevTools (F12)
2. Go to Application ‚Üí Storage
3. Clear localStorage
4. Refresh the page
